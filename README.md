# AI Finance Recommender System

## Project Description
The **AI Finance Recommender System** is a full-stack web application developed as a Final Year Engineering Capstone Project. The system analyzes user income, expenses, and savings to provide intelligent investment recommendations. It uses AI/ML techniques to classify stocks and mutual funds based on risk levels and integrates real-time market data to deliver personalized financial insights through a user-friendly web interface.

---

## Features
- Income and expense analysis
- Automatic savings calculation
- Stock recommendations (High Risk / Low Risk)
- Mutual fund recommendations (SIP and One-time investments)
- AI-based risk classification
- Real-time market data integration
- AI chat assistant using Google Gemini
- Responsive and user-friendly UI

---

## Tech Stack

### Frontend
- HTML
- CSS

### Backend
- Node.js / Python (based on implementation)

- Python

## Steps to run the Project : 

### Step 1 : Clone the Repository : 
```sh
https://github.com/MdAyan15/Invest-IQ-AI-Finance-Recommender
```

### Step 2 : Configure Environment Variables : 
Create a .env file in the root directory and add the following line:
```sh
GEMINI_API_KEY=your_google_gemini_api_key_here
```

**IMPORTANT** :
You must replace "your_google_gemini_api_key_here" with your own Google Gemini API key before running the project.

### Step 3 : Run the Proeject : 
```sh 
python main.py
```

### Output 
The result would be displayed on the local link 
```sh
* Running on http://127.0.0.1:5000
```
